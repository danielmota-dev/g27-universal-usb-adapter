# G27 Universal USB Adapter

## Overview
Universal USB adapter developed in C++ for Logitech G27 pedals and shifter, supporting both original Logitech hardware and custom-built (DIY) pedals and shifters.

The system supports multiple operating modes and performs automatic configuration based on the connected hardware.

## Key Features
- USB HID device
- Support for Logitech G27 pedals and shifter
- Support for custom (DIY) pedals and shifters
- Automatic hardware detection
- Multiple operating modes
- Modular and scalable architecture

## Operating Modes
1. Logitech pedals + Logitech shifter  
2. Pedals only (Logitech or DIY)  
3. Logitech shifter only  
4. DIY shifter with automatic configuration  

## Technologies
- C++
- Arduino (ATmega32u4)
- USB HID

## Hardware Compatibility
- Logitech G27 pedals
- Logitech G27 shifter
- Custom pedals
- Custom shifter
